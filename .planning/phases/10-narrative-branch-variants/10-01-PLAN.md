<interfaces>
Branch profile shape (D-01, D-02):
```typescript
interface BranchTransitionOverride {
  fromTopic: string;
  toTopic: string;
  presetId: string;
  rationale: string;
}

interface AssemblyBranch {
  id: string;
  label: string;
  sequence: string[];
  transitionOverrides?: BranchTransitionOverride[];
}

interface LongFormAssemblyProfile {
  schemaVersion: "1.0.0";
  slug: string;
  sequence?: string[];           // required when branches absent
  branches?: AssemblyBranch[];   // required when sequence absent
  defaultBranchId?: string;      // required when branches present
  targetWindowMinutes: { min: number; max: number };
  defaultPacingPresetId?: string;
}
```

Schema slug pattern: extend to allow `-branched-v` suffix (e.g. `content-depth-branched-v1`).

New fork presets (D-05):
- `auth-session-to-mitm-defense` (attack-path: skip pki)
- `mitm-defense-to-oauth-jwt-session` (attack-path: skip zero-trust)
- `pki-trust-chain-to-zero-trust-access` (defense-path: skip mitm)

Stitched sceneId for branches: `${slug}:${branchId}` passed to stitchSceneSpecsInOrder.
