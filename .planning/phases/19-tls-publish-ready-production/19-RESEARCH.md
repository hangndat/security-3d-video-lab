## Production SceneSpec Architecture

```
tls-production-scene-spec.json
  actors: client, server
  packets: hook-threat, client-hello, server-hello, app-data (routes per beat)
  timeline: 5 packet cues aligned to contract startFrame/endFrame windows
  totalFrames: 236
```

Validation gates:
- `validateSceneSpec` pass
- Each contract `storyboardBeats[].id` has matching timeline cue id substring
- `computeTopicFrameSpan(tls contract, scene)` ≥ contract max endFrame

## Viz-Aware Render Trace

```
deriveRenderFrameState(sceneSpec, frame)
  → scheduleFrame + buildVizFrameState (existing)
  → getComposePlan(sceneSpec, frame, { captionMap })
  → vizRenderTraceInput = seed:frame:renderOrder:packetPositionsHash
  → colorFromTrace(vizRenderTraceInput) for PPM frames
```

Determinism: same scene + frame + caption map → identical trace and MP4 hash across runs.
